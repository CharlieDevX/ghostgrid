# GhostGrid Appearance Preferences

## Color Palette

| Role       | Hex       | Notes                          |
|------------|-----------|--------------------------------|
| Background | `#0d0f14` | Main page background           |
| Surface    | `#161b24` | Widget/card background         |
| Border     | `#252d3d` | Widget borders, dividers       |
| Accent     | `#7aa2f7` | ✅ Preferred blue — headings, links, bars, buttons |
| Green      | `#9ece6a` | Running/success states         |
| Red        | `#f7768e` | Danger/stopped states          |
| Text       | `#c0caf5` | Primary body text              |
| Muted      | `#565f89` | Labels, secondary info         |
| Warn       | `#e0af68` | Warning state (70–89% usage)   |

## General Style

- **Theme:** Dark, inspired by Tokyo Night
- **Border radius:** `10px` on widgets, `6px` on inputs/buttons
- **Font:** `Segoe UI`, system-ui fallback
- **Base font size:** 14px

## Notes

- The accent blue `#7aa2f7` is the hero color — used for headings, links, progress bars, and primary buttons. Keep this consistent across all widgets.
